Site de peças de carro
# Autoparts
page online para compra de peças de carro
uma loja online  de autoPeças

# O que o projeto faz:
uma pagina com carros da loja fisica. é possivel
visualizar os carros disponiveis e ao clicar no carro escolhido ver as peças disponiveis no estoque.

# Por que o projeto é útil:
ajuda a aumentar as vendas online.

# Quem mantém e contribui com o projeto :
feito com HTML e Css e bootstrap por mim mesmo, ainda terá atualização com implementaçoes novas afim de melhorar a experiencia do úsuario
como comprar pela propria plataforma, por enquanto o site manda para finalizar a compra no mercado libre.
